## 1. EVENT STRUCTURE

```typescript
interface Event {
  type: 'CHILD_RESPONSE' | 'CHILD_INACTIVE';
  correct?: boolean;                    // Answer evaluation result
  response?: string;                    // Child's transcribed speech
  previousResponse?: string;            // For repetition detection
  previousPreviousResponse?: string;    // For triple repetition
  signals?: {                           // Pre-detected by voice layer
    screaming?: boolean;
    crying?: boolean;
    prolongedSilence?: boolean;
  };
}
```

**Event Sources:**
| Event Type | Trigger |
|------------|---------|
| `CHILD_RESPONSE` | Child speaks (transcription completed) |
| `CHILD_INACTIVE` | Inactivity timer fires (GREEN=30s, YELLOW=25s, ORANGE=20s, RED=15s) |
